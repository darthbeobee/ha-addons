# darthbeobee's Home Assistant Add-ons

Welcome to my custom repository for Home Assistant OS Add-ons! 

To install any of these in your Home Assistant instance:
1. Go to **Settings > Add-ons > Add-on Store**
2. Click the three dots (`...`) in the top right and select **Repositories**
3. Add this URL: `https://github.com/darthbeobee/ha-addons`

## Available Add-ons

* **[Kinect V2 Motion Watcher](./kinect_motion/)**: Transforms an Xbox One Kinect into a highly sensitive, local-only MQTT motion sensor optimized for low-power hardware.
